GUARDD: User-friendly MATLAB software for rigorous analysis of relaxation-dispersion NMR data.
==============================================================================================

Required Software
-----------------

-  `MATLAB <http://www.mathworks.com>`__
-  `MATLAB Optimization
   Toolbox <http://www.mathworks.com/products/optimization/>`__
-  `MATLAB Curve Fitting
   Toolbox <http://www.mathworks.com/products/curvefitting/>`__
-  `MATLAB Statistics
   Toolbox <http://www.mathworks.com/products/statistics/>`__
-  `GUARDD <http://code.google.com/p/guardd/downloads/list>`__
-  Matlab .m and .fig files

Notes
-----

- The GUARDD tutorial walks users through the basic features of the program 
- The GUARDD manual contains practical and theoretical considerations of relaxation-dispersion NMR
